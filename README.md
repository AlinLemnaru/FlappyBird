# FlappyBird

## Description
A 2D FlappyBird-style game built in Unity.  
Test your reflexes by navigating the bird through the pipes! This project demonstrates:

- Unity 2D development  
- C# scripting  
- Game mechanics like collisions, scoring, and procedural obstacles  

## Features
- Endless runner gameplay  
- Score tracking  
- Increasing difficulty over time    

## Tech Stack
- Unity 
- C#  
- Visual Studio
